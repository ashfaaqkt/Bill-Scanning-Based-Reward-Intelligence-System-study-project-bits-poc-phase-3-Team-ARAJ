What is this system?

The Bill Scanning Reward AI System is an innovative Proof of Concept leveraging cutting-edge Gemini AI to digitize, analyze, and process retail receipts. It instantly extracts line-item data, dynamically validates duplicate scans, and calculates intelligent reward multipliers based on shopping categories and user streak behavior. It transforms routine shopping into an engaging, gamified experience while ensuring privacy and enterprise-level data handling.

About This PoC

This is a Proof of Concept simulation for the BITS Pilani final-year project powered by Gemini AI. It validates the UI flow and real ML/Reward concepts with a live backend.

Current Limitations:
  • Fraud detection is simplified.
  • Full user segmentation is not yet implemented.
  • DSP export is simulated.
  • OCR is powered by Google Gemini AI.
